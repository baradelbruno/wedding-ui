# Wedding UI

A React application built with Vite.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Tech Stack

- React 18.3
- Vite 5.4
- Modern JavaScript (ES6+)
